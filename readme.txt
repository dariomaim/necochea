Necochea theme. A diary for life surfers.

== Description ==

Necochea theme is a fast and lightweight theme for writers and surfers. It was built with the Gutenberg editor in mind and has a lot of options that makes it extendable and customizable.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Type in Necochea in the search form and press the 'Enter' key on your keyboard.
3. Click Activate to use your new theme right away.

All of our options can be found in the Customizer in 'Appearance > Customize'.
